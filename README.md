# FSW Trips
- Projeto desenvolvido no curso Full Stack Week by Felipe Rocha.
---------------------------

## Tecnologias 
- Next.js
- PrismaORM
- Tailwind
- GoogleOAuth
- TypeScript
------------------------------

## Getting Started
- npm install or yarn add

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
